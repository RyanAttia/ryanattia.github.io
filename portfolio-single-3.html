Programmable Water Leak Demonstration Unit

Project Date Jan - May, 2024
Role Data Analysis, Full-stack, Software Engineering
Client Washington Suburban Sanitary Commission (WSSC)

As part of a collaborative engineering team, I developed an interactive, real-time water leak demonstration system for the Washington Suburban Sanitary Commission (WSSC), one of the largest water utilities on the East Coast. The goal was to educate customers about the financial and environmental impact of toilet leaks through a hands-on, visual display.

I was responsible for building two core software components:

  • A Python-based program that runs on a Raspberry Pi, connected to an adapted water flow sensor. This system accurately detects and processes real-time leak data, including flow rate and leak severity.
  • A Java-based GUI, developed in NetBeans, that visualizes this data, simulating various types of toilet leaks. The interface displays water waste in real time and projects monthly and quarterly cost impacts. It also includes full support for Spanish-language users and customizable pricing models to align with different billing structures.

The hardware-integrated demonstration unit includes a functional toilet model that simulates different leak types. The system enables users to see exactly how much water and money is wasted over time — a compelling tool for public awareness and customer engagement. This project exemplified end-to-end system integration, cross-language communication between Python and Java, real-time data processing, and user-centered design. It was successfully deployed as a prototype for future use in WSSC’s customer outreach programs.
