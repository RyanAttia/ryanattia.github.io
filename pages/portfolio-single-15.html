Multithreaded Thread Scheduler with CPU and IO Device Management

Project Description

Developed a custom thread scheduler in C to simulate CPU and I/O device management in a simplified operating system environment. The scheduler supports concurrent task execution across a single CPU core and two I/O devices using distinct scheduling policies: Shortest Remaining Time First (SRTF) for CPU-bound tasks and First-Come, First-Served (FCFS) for I/O-bound tasks. The core of the system was built using the pthreads library to enable multithreading, with mutex locks, condition variables, and semaphores ensuring proper thread synchronization and preventing race conditions.

Threads represented individual processes that requested access to CPU and I/O resources, and the scheduler dynamically handled context switching and queuing based on current system load and remaining execution time.
