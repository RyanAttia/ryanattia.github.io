Calculator & Digital Clock with HCS12

Project Description

Wrote a program in Full Assembly using CodeWarrior IDE that combines an elementary calculator displayed on the HyperTerminal connected to the HCS12 board, and a digital clock displayed on 7-segment displays attached to PORTB and the HyperTerminal connected to the HCS12 board simulation.

The application supports a full 24-hour digital clock in hh:mm:ss format. Both the calculator and digital clock run simultaneously, demonstrating real-time multitasking in a low-level Assembly environment. The project showcases deep understanding of microcontroller architecture, timer-driven interrupts, and peripheral interfacing.

Project Date April 2024
Tech Stack Embedded Systems, Assembly Programming
